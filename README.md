# Basic Template for Full Stack Projects

## Frontend

- React
  - Vite
  - Tailwindcss

### Initialize

```bash
npm i 
```

### Start Dev to default port 5173

```bash
npm start dev
```

## Backend

- Python
  - FastAPI

## Database

- MongoDB

## Docker

- Backend and DB are managed by Docker compose
  - Backend port 8008
  - DB port 27018

```bash
docker compose up
```
